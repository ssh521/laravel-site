# Test contract

- menu button은 `aria-controls`와 `aria-expanded`를 가진다.
- Escape, backdrop, 내부 링크로 닫히고 menu button으로 focus가 돌아온다.
- modal menu는 focus trap과 body scroll lock을 제공한다.
- 고정 primary action은 768px 미만에서만 표시되고 safe-area inset을 반영한다.
- 320px에서 CTA 문구가 한 줄이며 가로 overflow가 없다.
- reduced motion에서는 panel transition이 제거된다.
